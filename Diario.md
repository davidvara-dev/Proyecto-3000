📖 La primera regla de oro de CSS
Regla #1

En CSS, no todas las propiedades se heredan.

Ejemplos:

✅ Se heredan:

color
font-family
font-size

❌ No se heredan:

background-color
border
margin
padding

Regla

Una propiedad de alineación solo puede mover un elemento si existe espacio disponible para moverlo.

Esta idea aparece en:

Flexbox
Grid
Posicionamiento
Incluso en algunos sistemas de interfaces gráficas de Java.
